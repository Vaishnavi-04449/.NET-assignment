This project is a simple C# console application that allows the user to enter details for multiple students and calculates their grades based on the marks entered.

The program demonstrates basic C# fundamentals, including:

Classes and objects

User input

Loops

Switch expressions

Console output
Features

Accepts details for multiple students

Stores student information such as Name, Age, and Marks

Calculates grade based on marks

Displays the result for each student
How the Program Works

The user enters the number of students.

The program runs a loop to collect details for each student.

The program calculates the grade using a switch expression.

The program displays the student information and grade.
