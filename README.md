Student Grade Calculator – C# Console Application
Overview

This project is a simple C# console application that allows the user to enter details for multiple students and calculates their grades based on the marks entered.

The program demonstrates basic C# fundamentals, including:

Classes and objects

User input

Loops

Switch expressions

Console output

Features

Accepts details for multiple students

Stores student information such as Name, Age, and Marks

Calculates grade based on marks

Displays the result for each student

Grade Criteria
Marks	Grade
90 and above	A
75 – 89	B
50 – 74	C
Below 50	Fail
How the Program Works

The user enters the number of students.

The program runs a loop to collect details for each student.

The program calculates the grade using a switch expression.

The program displays the student information and grade.
Fibonacci Series – C# Console Application
Overview

This project is a simple C# console application that generates and displays the Fibonacci series based on the number of terms entered by the user.

The Fibonacci series is a sequence of numbers where each number is the sum of the previous two numbers.

Fibonacci Series Example
0 1 1 2 3 5 8 13 ...

Each number is calculated as:

Next Number = Previous Number + Current Number
Features

Accepts number of terms from the user

Generates the Fibonacci sequence

Displays the series in the console

Demonstrates use of loops and variables in C#

How the Program Works

The user enters the number of terms.

The program initializes the first two numbers of the Fibonacci series:

0

1

A loop is used to calculate the next numbers.

Each number is printed to the console
